# FitVision: AI Virtual Try-On

## 🚀 Project Overview
FitVision is a web application that allows users to virtually "try on" clothes. By uploading a photo, users can see how different garments from online catalogs look on their own body using generative AI.

## 🛠 Tech Stack
- **Frontend:** React
- **Backend:** Flask (Python)
- **Database:** PostgreSQL (AWS RDS)
- **File Storage:** AWS S3
- **AI Model:** Fashn.ai
